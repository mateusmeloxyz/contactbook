# ContactBook - Full Stack CRUD App

## Dependencies:

* `docker`
* `docker compose`

## Run instructions:

Clone the repository to your machine, then run the following in the project root folder:

```docker compose up -d --build```

## Viewing the application

The frontend will be diplayed on [localhost:80](http://localhost:80), and the backend swagger interactable panel at [localhost:8080/swagger](http://localhost:8080/swagger)
